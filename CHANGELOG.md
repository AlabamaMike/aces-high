# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Core ECS architecture with component system
- Basic game state management
- Collision detection system with spatial hashing
- Weapon system with multiple fire patterns
- Performance monitoring and profiling tools
- WebGL shader system with WebGL 1.0/2.0 support
- Comprehensive test suite
- Build pipeline with webpack and wasm-pack
- Documentation (README, LICENSE, CONTRIBUTING)

### In Progress
- AI system for enemy behaviors
- Procedural generation system for zones and waves
- Upgrade system with synergies
- Complete aircraft implementations (5 types)
- Renderer system with sprite batching
- Audio system with spatial sound
- Input handling for keyboard, mouse, and gamepad
- Web bindings for browser APIs
- Asset loading and management

### Planned
- Meta-progression system
- Save/load functionality with IndexedDB
- UI system and menus
- Particle effects system
- Boss battles
- Achievements system
- Leaderboards
- Mobile touch controls
- Accessibility features (colorblind modes, etc.)
